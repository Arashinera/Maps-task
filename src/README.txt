Map
-----
Створіть проект Maps-task, де реалізуйте
відповідний функціонал.
+
В магазинчику є певний набір з 4-х різних товарів.
Product має властивості: найменування, кількість, ціна.
Властивості є різними за значеннями для кожного товару.
+
За допомогою Map сформуйте надання всього набору товарів
(у вигляді Map, де id є ключем, а об'єкт товару - значенням),
виведення цього набору у вигляді нумерованого переліку.
+
Також, реалізуйте виведення одного товара за певним id,
отримання якого забезпечте через Scanner.
+
Забезпечте розподілення функціоналу проекту за різними
відповідними класами.
+
